===============
@script_panel
===============

.. image:: @script_panel-UI.png

.. image:: @script_panel-ex.png

ScriptUIを実装したスクリプトを読み込む。

使い方
--------------------
**O** ボタンを押すと、あるフォルダが開くので、そこのフォルダにScriptUIを実装したスクリプトを放り込む（フォルダ内の内容がスクリプト起動時と変わった場合は、 **R** ボタンを押すと再読み込みされる）。

後は、ドロップダウンリストから適当なスクリプトを選択する。

挙動は、当然対象のスクリプトがどう実装されているかに依存する。
