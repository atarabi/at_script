===============
@movie_maker
===============

.. image:: @movie_maker-UI.png

コンポ、レイヤーの画を動画(GIF, APNG)として保存する。


解説
--------------------

ファイルパス
^^^^^^^^^^^^^^^^^^^^

.. image:: @movie_maker-path.png

保存先となるファイルパスを指定する。 **[name]** はコンポ名もしくはレイヤー名に、 **[ext]** は指定した画像形式に対応した拡張子に、 **##** はインデックス番号に、保存時にそれぞれ置換される。

画像形式
^^^^^^^^^^^^^^^^^^^^

.. image:: @movie_maker-ext.png

画像形式を選択する。

対象
^^^^^^^^^^^^^^^^^^^^

.. image:: @movie_maker-target.png

対象をコンポジションか選択レイヤーかで選ぶ。

範囲
^^^^^^^^^^^^^^^^^^^^

.. image:: @movie_maker-area.png

動画にしたい範囲をワークエリア、コンポジション全体から選択する。

解像度
^^^^^^^^^^^^^^^^^^^^

.. image:: @movie_maker-res.png

解像度を **Full**、 **Half** ……から選択する。

スキップ
^^^^^^^^^^^^^^^^^^^^

.. image:: @movie_maker-skip.png

何フレームおきに出力するかを決定する。

スピード
^^^^^^^^^^^^^^^^^^^^

.. image:: @movie_maker-speed.png

何倍速にするかを決める。

出力
^^^^^^^^^^^^^^^^^^^^

.. image:: @movie_maker-make.png

**Make** ボタンを押して出力。
