Clipboard¶

Clipboard.getText()¶

getText(): string;

クリップボードにある文字列を取得する。

  • TypeScript
  • JavaScript
const text = Atarabi.clipboard.getText();

var text = Atarabi.clipboard.getText();

Clipboard.setText()¶

setText(text: string): void;

クリップボードに文字列を設定する。

  • TypeScript
  • JavaScript
Atarabi.clipboard.setText('wiggle(3, 100)');

Atarabi.clipboard.setText('wiggle(3, 100)');
